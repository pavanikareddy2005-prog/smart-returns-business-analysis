# Smart Returns & Customer Trust Platform

A complete Business Analyst portfolio project for an e-commerce return-management platform.

## Problem
30% rise in returns -> high costs, slow refunds, fraud, poor communication.

## Solution
Analytics-driven return platform: self-serve returns, eligibility engine,
fraud scoring, auto-refunds (<=2 days), and BI dashboards.

## Key Results (Targets)
- Return rate 18% -> 13%
- Refund time 6d -> 2d
- Fraud 12% -> 7% | CSAT 62% -> 85%

## Skills Demonstrated
BRD/FRS, Agile, SQL, Power BI, Wireframes, BPMN/UML, Testing/UAT.
